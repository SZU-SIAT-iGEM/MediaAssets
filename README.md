### 操作指南

#### 图片上传

| 步骤 | 说明 |
|------|------|
| 1. 选择分类 | 根据图片类型选择对应文件夹 |
| 2. 命名文件 | 格式：`YYYY-MM-DD_描述_序号.格式`<br>示例：`2026-01-15_电泳图_01.png` |
| 3. 检查大小 | 确保单个文件 < 25MB |
| 4. 上传并注明 | Commit 信息写明图片内容和关联 Issue |

#### 视频索引创建（大文件索引）

由于视频文件较大（GitHub 对大文件支持不佳），我们采用 **索引文件** 方式管理：

1. 将视频上传至网盘（百度网盘/阿里云盘/Google Drive）
2. 在 `视频索引/` 创建同名 `.txt` 文件
3. 使用 `templates/视频索引模板.txt` 填写信息
4. 对于可能的其他大文件，对视频索引修改后建立索引。 

**视频索引必填内容（例，如有）**：
- 视频标题
- 存储平台与链接
- 提取码/密码
- 视频简介

### 图片质量要求

| 用途 | 格式 | 分辨率要求 | 说明 |
|------|------|------------|------|
| 实验数据图 | PNG/TIFF | 原始分辨率 | 不压缩，保持原始质量 |
| Wiki 展示 | PNG/JPG | ≥ 1200px 宽 | 可适度压缩 |
| 社交媒体 | JPG | 按平台要求 | 可压缩以加快加载 |
| 海报打印 | TIFF/PNG | 300 DPI | 高分辨率 |
